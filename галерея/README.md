# Галерея

Blueprint `gallery`. Витрина, каталог картин, профили художников, аукционы, загрузка картин, избранное, скачивание оригиналов.

## Маршруты
- `/` — витрина
- `/catalog` — каталог
- `/painting/<id>` — страница картины
- `/artists` — список художников
- `/artist/<id>` — профиль художника
- `/auctions` — аукционы
- `/auctions/<id>/bid` — ставка на аукцион
- `/cabinet/upload` — загрузка картины (художник)
- `/download/<id>` — скачать оригинал
- `/favorite/toggle` — добавить/убрать из избранного
